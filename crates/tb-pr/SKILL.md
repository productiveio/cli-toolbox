# tb-pr

GitHub PR radar — a kanban TUI + non-interactive CLI for tracking all PRs that need your attention across the Productive GitHub organization.

## Status

Skeleton only. Data fetching, cache, and TUI land in later milestones. This SKILL.md is a placeholder that ships with M1.

## Quick reference

```bash
tb-pr doctor          # verify gh auth + config
tb-pr config init     # write default config
tb-pr prime           # context dump (stub until later milestones)
```

## Getting started

Run `tb-pr prime` for available commands. Run `tb-pr doctor` to verify setup.
