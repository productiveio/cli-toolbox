# tb-prod

CLI for interacting with the Productive.io API. Manages tasks, todos, comments, and project context with compact, AI-optimized output. Built for AI agent consumption but works for humans too.

## Capabilities

- **Tasks** — list, create, update, and inspect tasks with filters (project, assignee, status, search)
- **Todos** — create and manage subtask checklists within tasks
- **Comments** — add comments to tasks (supports stdin and file input)
- **Projects** — view project structure, statuses, and task lists
- **Cache** — local sync for fast lookups of people, projects, and statuses

## Getting started

Run `tb-prod prime` for a command reference and current configuration context.
Use `tb-prod <command> --help` for detailed command usage.
